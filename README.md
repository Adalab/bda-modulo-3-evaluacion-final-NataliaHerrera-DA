Evaluación de Actividades de Vuelo

### **Objetivo**

Este documento sirve como un **registro fundamental** de las actividades de vuelo evaluadas para el cliente.

---

### **Estructura de Datos Clave**

La evaluación se basa en un conjunto de datos que incluye información esencial, como la siguiente columna:

* **Year (Año):** Indica el **año** en el cual se registraron las actividades de vuelo para el cliente.
